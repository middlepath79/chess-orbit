# ♔ Orbit Chess — The Living Chess Universe

**Drag. Click. Explore. Learn.**

Orbit Chess transforms the classic chess blog into an immersive 3D solar system. Six beautifully crafted chess pieces orbit a glowing sun — each one a gateway to something meaningful:

- ♕ **Play** — Lichess & Chess.com
- 📜 **Learn** — 1500 years of chess history
- 🧠 **Grow** — Cognitive benefits backed by NIH research
- ♟️ **Experiment** — Chess variants (Crazyhouse, Chess960)
- 📰 **Stay Updated** — Live tournaments with prize pools

## ✨ Why It's Cool

- **No clicks wasted** — Every piece has a purpose
- **Always fresh** — Tournament news auto-updates
- **Professional polish** — Loading screen, toast notifications, responsive footer
- **Zero dependencies** — Single HTML file, runs anywhere

## 🎮 Built With

Three.js • JavaScript ES6 • CSS3 Animations • Chess.com & Lichess APIs
